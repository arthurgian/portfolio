## Começando

### Pré-requisitos

- Node.js 18+
- npm 9+

### Instalação

1. Clonar repositório:
   ```sh
   git clone https://github.com/arthurgian/portfolio.git
   ```
2. Instalar dependências:
   ```sh
   npm install
   ```
3. Iniciar servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

### Build de produção

```sh
npm run build
```
